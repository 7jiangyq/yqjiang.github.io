# yqjiang.github.io

Personal homepage repository for GitHub Pages.

## Publish

1. Create the repository `yqjiang.github.io` under the GitHub account `7jiangyq`.
2. Push this repository to `https://github.com/7jiangyq/yqjiang.github.io`.
3. In GitHub repository settings, enable GitHub Pages from the default branch root if it is not enabled automatically.

## Edit

- Main page: `index.html`
- Styles: `assets/style.css`

Replace the placeholder biography, education, projects, publications, and contact information with your own content.
